# Session Logger

LLMとの会話内容をObsidian Vault内のLLM/ディレクトリに保存するスキルです。

## 保存ルール

1. **保存先**: `LLM/` ディレクトリ
2. **ファイル名**: `YYYY-MM-DD_HHmm_[サマリ].md`
   - 例: `2025-01-15_1430_プラグイン公開準備.md`
3. **内容の構成**:
   ```markdown
   # [ユーザーの依頼内容のサマリ]

   - 日時: YYYY-MM-DD HH:mm
   - ディレクトリ: [作業ディレクトリ]

   ## 会話内容

   ### ユーザー
   [ユーザーの入力]

   ### Claude
   [Claudeの応答]

   ## 作成・編集したファイル

   - [[path/to/file1.md]]
   - [[path/to/file2.ts]]
   ```

## 使用方法

ユーザーが以下のような発言をした時に自動的に発動:
- 「このセッションを保存して」
- 「ログを残して」
- 「会話を記録して」

または `/save-session` コマンドで明示的に実行。

## 注意事項

- Obsidian形式のリンク `[[filename]]` を使用
- 作成・編集したファイルは必ずリンクとして記録
- LLM/ディレクトリが存在しない場合は作成する
