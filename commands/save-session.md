# セッション保存

このコマンドは現在の会話内容をLLM/ディレクトリに保存します。

## 実行手順

1. LLM/ディレクトリの存在を確認（なければ作成）
2. ファイル名を生成: `YYYY-MM-DD_HHmm_[サマリ].md`
3. 以下の内容でファイルを作成:

```markdown
# [ユーザーの依頼内容のサマリ]

- 日時: [現在日時]
- ディレクトリ: [作業ディレクトリ]

## 会話内容

[会話のサマリと重要なポイント]

## 作成・編集したファイル

[Obsidian形式のリンクでファイルを列挙]
- [[path/to/file]]
```

4. 保存完了をユーザーに報告

## 注意

- 会話全文ではなく、重要なポイントをサマリとして保存
- 作成・編集したファイルは必ずリンクとして記録
- 日本語でサマリを作成
